# 한국 커뮤니티 홍보 템플릿 (복붙용)

본인이 만든 서비스임을 투명하게 밝히면서, 가치를 먼저 주는 방식으로 구성됨.
게시 전 체크: (1) 본문 80%+ 가 정보가치 있는지 (2) 출처 명시했는지 (3) 해당 커뮤니티 정책 확인했는지.

## 🟢 네이버 지식인 답변 템플릿

### 질문 유형: "일본 4박5일 경비 얼마나 들까요?"

```
도쿄 기준으로 실제 블로그 후기 기반 평균 데이터 공유드립니다.

【4박 5일 총경비】
🎒 알뜰 (호스텔+현지식): 약 95만원
🏨 표준 (비즈니스호텔): 약 140~150만원
💎 프리미엄 (4성+): 280만원 이상

【구성 비율】
- 항공 왕복: 35~45만원
- 숙박 4박: 40~60만원 (비즈니스)
- 식비: 1일 4~6만원
- 지하철·JR: 1일 5천~1만원
- 입장료·쇼핑: 10~20만원

【팁】
· 후쿠오카는 도쿄 대비 20~30% 저렴
· 성수기(4월 벚꽃/8월/연말) 항공권 2배 가까이 오름
· 환전 공항 X, 트래블월렛 추천

제가 만든 여행 경비 계산 사이트(트립비, www.tripbudget.my)에서 
날짜·인원 입력하면 성수기 반영한 실시간 계산이 나옵니다. 
참고하세요 :)
```

### 질문 유형: "20대 첫 해외여행 어디가 좋을까요?"

```
예산과 목적에 따라 다른데, 예산별로 현실적인 추천드립니다.

【100만원 이하】
베트남 다낭 (3박4일) — 85~95만원
· 항공 35만, 4성 호텔 1박 7만, 음식 저렴

【100~130만원】
태국 방콕 or 일본 후쿠오카 (4박5일)
· 방콕: 인프라 최강, 음식 다양
· 후쿠오카: 가까움, 편리

【130~180만원】
일본 도쿄·오사카 or 대만 일주 (4박5일)

【동남아 5국 가성비 순위】
베트남 > 필리핀 > 태국 > 대만 > 싱가포르

디테일한 계산은 제가 운영하는 트립비(www.tripbudget.my)에서
국가별·기간별 나오니 참고하세요. 본인 만든 거라 투명하게 밝힙니다.
```

## ❗ 절대 하면 안 되는 것

- 부계정 여러 개 만들어서 "이 사이트 좋아요!" 셀프 댓글
- 본인이 만든 걸 숨기고 "우연히 찾았다" 식으로 홍보
- 안 가본 여행을 간 것처럼 가짜 후기 작성
- 같은 글 여러 커뮤니티에 토씨 없이 복사 (어뷰징 탐지)
- 네이버 검색 키워드를 부자연스럽게 본문에 반복

들키면 사이트 자체가 "광고사이트"로 낙인찍혀서 회복 불가.

## 📅 권장 실행 플랜 (4주)

**1주차**: 네이버 지식인 답변 하루 3개씩 (여행 경비/예산 질문만)
**2주차**: 인스타 릴스 첫 3개 업로드 + 본인 블로그 시작
**3주차**: 네이버 블로그 글 2개 포스팅 + 클리앙 자유게시판 소개글
**4주차**: 카페 운영진 쪽지 시작 + 첫 정보글 시도
